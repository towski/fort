There is something I don't like about using jQuery, and I think I'm starting to see what it is. The library makes it hard to know what is built-in to the language, and what is part of jQuery. Also decided to build my own simple ORM using Berkeley Databases, for Thoughts on my homepage. I figure they might be a great solution for a mostly read-only environment, with only one writer. They can handle concurrency, so multiple unicorns. And they just seem simpler than sqlite or mysql, especially for a small self-hosted website. I am using the DBDB gem . Code for that is in fort, but I'll probably split it off into it's own repository. I am reminded that the most important reason to build things yourself is to learn.
