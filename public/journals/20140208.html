Also my go ruby bytecode interpretter is further along now. I enjoyed it more when I realized that every object in my ruby runtime could be an Object pointer. For example: the Class Object is instantiated and sent as the class of every new class object:


CLASS = &Object{}
IO = NewObject(CLASS, "IO")

For now each class has a map for storing references to it's functions. Here is the code for defining the internal ruby method IO#puts:


IO.methods[":puts"] = func(v ...*Object) *Object {
...
}

It's a function that takes any number of Object pointers, and returns an Object pointer. With these tools, I can now parse class definitions and method definitions. Up next, handling blocks, and probably some code reorganization.
